ActiveMQ Artemis 1.5.5 Release Notes

A complete list of JIRAs for the 1.5.5 release can be found at the Apache ActiveMQ Artemis project JIRA.

Bug

  • [ARTEMIS-930] - JDBC FileFactory Driver Prepared Statement does not always return ID
  • [ARTEMIS-990] - AMQ119032: User: Customer does not have permission=’CREATE_DURABLE_QUEUE’ on address $sys.mqtt.queue.qos2
  • [ARTEMIS-1033] - NPE on PageTransactions during startup in case of previous TX Failures
  • [ARTEMIS-1043] - NettyConnector not working with IPv6 address
  • [ARTEMIS-1057] - Lower log level of AMQ214023
  • [ARTEMIS-1071] - Invalid Type exception handling improvements
  • [ARTEMIS-1072] - ./artemis create –java-opts will have options on bin instead of etc
  • [ARTEMIS-1074] - Acceptors and Connectors that enable SSL don’t start up while using a password codec
  • [ARTEMIS-1078] - Thread pool executor occasionally misses to create a new thread
  • [ARTEMIS-1089] - Backup sync-up is slow when paging
  • [ARTEMIS-1114] - Journal Compactor possibly losing records
  • [ARTEMIS-1140] - Avoid Queue lock on queueQuery

Improvement

  • [ARTEMIS-1077] - Improving InVM and Buffer usage
  • [ARTEMIS-1124] - JDBC Network Timeout configuration
